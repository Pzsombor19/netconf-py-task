# NetConf Python Task

Ez a projekt egy egyszerű Python programot tartalmaz, amely egy NetConf szerverhez csatlakozik, és alapvető NetConf műveletek végrehajtását teszi lehetővé.

## Funkcionalitás
- A program futáskor bekéri a NetConf szerver IP címét
- NetConf kapcsolódás létrehozása
- Konfiguráció lekérdezése a szerverről
- Egyszerű, könnyen érthető felépítés

## Követelmények
- Python 3.x
- ncclient Python csomag
